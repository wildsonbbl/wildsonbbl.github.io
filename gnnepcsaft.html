GNNePCSAFT

This app uses a GNNePCSAFT model to estimate PC-SAFT parameters. Parameters can be used with FeOs package, DWSIM or any other implementation of the EoS.

Web App is offline for now, considerer downloading the Desktop App (Windows, Mac, Linux).

Follow the link bellow to SourceForge and install the app. More info about the project at github repo.
